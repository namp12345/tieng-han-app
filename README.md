# K-Tour Guide v6 PWA

Đây là bản web app/PWA sẵn sàng để đưa lên GitHub Pages.

## Cách dùng nhanh
1. Upload toàn bộ file trong thư mục này lên repo GitHub.
2. Bật GitHub Pages cho branch chính.
3. Mở link Pages bằng Chrome trên Android.
4. Chọn **Cài đặt ứng dụng** hoặc **Thêm vào màn hình chính**.
